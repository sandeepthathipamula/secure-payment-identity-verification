# Secure Payment Identity Verification System

A web-based prototype security system designed to improve the safety of mobile payment applications.

## Features
- User-defined transaction limit
- Identity verification for high-value payments
- Automatic photo capture for unauthorized attempts
- Temporary verification link sent to the user
- Image auto-deletion for privacy protection

## Tech Stack
- Node.js
- MongoDB
- HTML
- CSS
- JavaScript
- EJS

## Concept
The system introduces an additional security layer for mobile payments by verifying transactions that exceed a predefined limit.
